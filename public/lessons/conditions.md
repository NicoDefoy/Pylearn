# Conditions (if, else, elif)

Les conditions permettent d’exécuter du code seulement si une certaine situation est vraie.

```python
age = 18
if age >= 18:
    print("Majeur")
else:
    print("Mineur")
```

On peut aussi utiliser `elif` pour plusieurs cas :

```python
note = 15
if note >= 16:
    print("Très bien")
elif note >= 10:
    print("Moyen")
else:
    print("Insuffisant")
```

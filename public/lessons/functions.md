# Fonctions

Une fonction permet de regrouper du code réutilisable. On la définit avec `def`.

```python
def saluer(nom):
    print(f"Bonjour, {nom}!")

saluer("Alice")  # Affiche: Bonjour, Alice!
```

On peut aussi retourner une valeur :

```python
def addition(a, b):
    return a + b

resultat = addition(2, 3)  # 5
```
